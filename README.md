# Run the RX AI Study Platform locally

This project contains a Vite + React frontend and an Express server for the study platform.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Start the development server:
   `npm run dev`
3. Open the app:
   `http://localhost:3000`

## Production Build

1. Build the app:
   `npm run build`
2. Start the production server:
   `npm run start`
